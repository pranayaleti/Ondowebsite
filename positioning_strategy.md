## Visual & UX Direction
| Element | Recommendation |
| :--- | :--- |
| **Typography** | **Heading**: Bold, high-contrast sans-serif (e.g., Montserrat or custom). **Body**: Clean, highly readable sans-serif (e.g., Inter). |
| **Color Palette** | **Primary**: Deep Charcoal (#121212). **Accent**: Vibrant International Orange (#FF4F00). **Neutral**: Pure White (#FFFFFF) and Slate Gray (#707070). |
| **Motion** | Subtle parallax on hero text, smooth fade-in transitions for sections, and hover-state scaling for interactive cards. |
| **Imagery** | Abstract technical illustrations, high-fidelity dashboard mockups, and minimal, professional photography of collaborative environments. |
